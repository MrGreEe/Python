Структура

Тег body(тело) - все помещенное внутрь этого элемента, отображается в основномокне браузера. Тег head(голова) - этот тег предшествует тегу(элементу) body, он содержит инфу о самой странице. В элемент head также часто включается элемент title - содержание этого тега вывоится либо в заголовке окна браузера, либо в качестве названия вкладки.

HTML(HyperTextMarkupLanguage) - язык разметки гипертекста

Текст

Блочные эементы - начинаются с новой строки Примерами могут быть элементы h l, р и div.

Встроенные элементы - располагаются в тексте и не начинаются с новой строки. Примеры таких элементов: Ь, i, img, em и span.

Формы

Все элементы формы помещаются в тег form. Для него всегда должен быть указан атрибут action, кроме того, для этого элемента часто устанавливают атрибуты method, id

    Типы (type - атрибут) форм бывают:

  1. text - когда атрибуту type присваивается значение text, это значит, что будет создано однострочное поле ввода текста
  2. password

Значения атрибута type :
text password textarea checkbox

Цвет

color (Основной цвет) - Свойство позволяет указать цвет текста внутри элемента. В CSS можно задать любой цвет одним из трех способов.

background-color (Цвет фона) - Каскадная таблица стилей обращается с каждым НТМL элементом так, будто он заключен в блок, а свойство background- color задает цвет фона этого блока.
